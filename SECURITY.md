# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x.x   | ✅        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via
[GitHub's private vulnerability reporting](../../security/advisories/new) or by
emailing the maintainer directly (see the package on
[jsr.io/@dostarora97/html2md](https://jsr.io/@dostarora97/html2md)).

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix

You will receive a response within 72 hours. Once a fix is released, the
vulnerability will be publicly disclosed with credit to the reporter (unless you
prefer to remain anonymous).
